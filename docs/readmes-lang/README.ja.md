# 🚀 PromptsManager

プロンプトを資産として管理しましょう。散らかったテキストではなく。

**PromptsManager**は、AIで使用するプロンプトを整理、カテゴライズ、再利用するためのアプリケーションです。
ChatGPT、Copilot、または他の生成モデルを使用している場合，你会发现優れたプロンプトが蓄積されていく一方で、、失われていきます。

このプロジェクトがその問題を解決します。

---

## 🧠 問題点

AIを Professionals:

* Notion、メモ帳、README、チャット履歴にプロンプトを散らかす
* すでに効果的なプロンプトを再作成する時間を無駄にする
* バージョン管理がない
* 蓄積された知識を構造化できない

良いプロンプトは記憶に頼るべきではありません。

---

## 🎯 解決策

PromptsManagerはプロンプトを一箇所に集約し、以下を可能にします：

* 📂 プロンプトの作成と保存
* 🏷️ カテゴリーとタグによる整理
* 🔎 素早い検索
* ♻️ 簡単な再利用
* 🧩 AI戦略の構造化

シンプルなアイデア：プロンプトを再利用可能なコードとして扱う。

---

## 🏗️ プロジェクト構造

```
/src
  /components      # Sidebar, PromptEditor, PromptList, MarkdownPreview, etc.
  /services        # storage.ts (localStorage + Electron IPC)
  /hooks           # useToast.ts
  /types           # Prompt.ts
  App.tsx          # メインコンポーネント
```

---

## ✨ 機能

- **完全な CRUD** - プロンプトの作成、編集、削除
- **カテゴリーとタグ** - 複数のタグでプロンプトを整理
- **お気に入り** - 好きなプロンプトをマーク
- **検索とフィルター** - カテゴリー、タグ、テキストでフィルター
- **並べ替え** - 日付（新しい/古い）、タイトル（A-Z/Z-A）、お気に入り順
- **自動保存** - 2秒間の非アクティブ後に自動保存
- **元に戻す/やり直し** - エディターでの編集履歴（Ctrl+Z / Ctrl+Y）
- **Markdown プレビュー** - フォーマットされたコンテンツを表示
- **テンプレート変数** - `{{variable}}` のサポートとハイライト
- **インポート/エクスポート** - JSON と Markdown
- **バックアップ** - 全データのエクスポート
- **レスポンシブインターフェース** - デスクトップとWebの両方で動作
- **デスクトップモード** - デスクトップ用の Electron バージョン

## 🚀 インストール

リポジトリをクローン：

```bash
git clone https://github.com/IamThiago-IT/PromptsManager.git
```

プロジェクトフォルダに移動：

```bash
cd PromptsManager
```

依存関係をインストール：

```bash
npm install
```

プロジェクトを実行：

```bash
npm run dev
```

### 利用可能なスクリプト

| スクリプト | 説明 |
|--------|-----------|
| `npm run dev` | 開発を開始 |
| `npm run dev:web` | Webのみモード |
| `npm run dev:desktop` | デスクトップモード (Electron) |
| `npm run build:web` | Web 用ビルド |
| `npm run build:desktop` | デスクトップビルド (Electron) |
| `npm run lint` | コードチェック |
